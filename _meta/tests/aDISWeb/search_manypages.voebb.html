So funktioniert das Filtern

Klicken Sie einen Filter an, um dessen Werte anzuzeigen.
Sie können Ihre Trefferliste einschränken, indem Sie einen oder mehrere dieser Werte von einem oder mehrere(n) Filtern auswählen und auf die Schaltfläche "Filtern" klicken. Sie können Ihre eingeschränkte Trefferliste wieder erweitern, indem Sie die Häkchen der Werte unter der Rubrik "Aktive Filter" wieder entfernen und die Schaltfläche "Filtern" auslösen.
